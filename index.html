Git

Git'as yra populiariausia kodo versijavimo programa.

Git komandos

  • init - sukuria git'o repozitoriją dabartiniame aplankale
  • status - nurodo informaciją apie lokalios git'o repozitorijos statusą
  • add - prideda nurodytus failus prie commit'ui paruoštų failų (stage'ina)
  • rm --cached - išima nurodytus failus iš stage'inimo zonos (unstage'ina)
  • commit -m "žinutė" - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutė turėtų būti trumpa (ne ilgesnė nei 50 simbolių) ir aiški)
  • restore - grąžina nurodytus failus į buvusio commit'o būseną (panaikina naujus pakitimus padarytus po paskutinio commit'o)
